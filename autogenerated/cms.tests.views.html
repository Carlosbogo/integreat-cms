Views¶

This package contains simple unit tests for views. It just ensures that all views return status code 200 and does not check whether the response content is valid in any way.

class cms.tests.views.view_test.ViewTest(methodName='runTest')[source]¶

Bases: django.test.testcases.TestCase

This test class is a helper class for other view tests. It does not contain tests on its own, but only helper functions for subclasses.

assertHttp200(response)[source]¶

This function checks whether the given response returns the http status code 200 and lets a test fail if not.

This function generates test functions for a list of views.
