RulesÂ¶

We use django-rules to add custom permissions for specific pages.

For a given user and page, the following permissions are added:

  • cms.edit_page if one of the following predicates return true:

    • can_edit_all_pages()

    • is_page_editor()

    • can_publish_all_pages()

    • is_page_publisher()

  • cms.publish_page_object if one of the following predicates return true:

    • can_publish_all_pages()

    • is_page_publisher()

See the projectâ€™s README to learn more.
