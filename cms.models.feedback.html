Feedback¶

This package contains all feedback-related data models: The base model Feedback as well as the following sub-models:

  • EventFeedback
  • EventListFeedback
  • OfferFeedback
  • OfferListFeedback
  • PageFeedback
  • RegionFeedback
  • SearchResultFeedback
